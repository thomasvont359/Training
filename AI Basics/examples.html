Bad → Better Prompt Examples

Bad

"Write a plan."

→

Better

"As a project manager, create a 5-step rollout plan for migrating X by Nov 30. Use bullets with owners/dates/risks, ≤150 words."

Bad

"Summarize this."

→

Better

"Summarize for execs (≤120 words), include 3 key risks and 2 decisions needed. Use bullets."

Bad

"Improve this email."

→

Better

"Rewrite for a frustrated customer; keep apologetic but confident tone; 100–150 words; include next steps and a 48-hour timeline."

Note the specificity: role, audience, constraints, and output format.
