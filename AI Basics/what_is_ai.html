What is AI?

Working definition + common types

Working definition

Software that learns patterns from data to make predictions, generate content, or decide actions, with outputs shaped by training data, instructions, and feedback.

Generative models

Draft, summarize, translate, create options (text/code/images/audio/video).

Predictive models

Forecast, score risk, route tickets (classification/regression).

Knowledge retrieval (search + cite)

Search internal docs and ground answers with citations.

Workflow automation

Connect tools and steps to complete tasks (file, transform, route, notify).
