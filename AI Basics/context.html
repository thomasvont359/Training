Context Is King

What to Include

Relevant Snippets Only

Paste only the lines that drive the task; cut intros and fluff. Highlight terms, metrics, or quotes the answer must preserve verbatim.

Specify Details

State audience, tone, region, deadlines, and acceptable terminology. Include reading level, taboos or terms to avoid, and one short example of preferred phrasing.

Source-of-Truth

Link the canonical doc, policy, or dataset. Quote exact passages to rely on, include version/date, and note which source wins if references conflict.

Instruct to Ask

If inputs are missing or ambiguous, ask targeted questions before proceeding. Offer default options and explain how outputs change based on each answer.
