Cheat Sheet

Prompt Starters

Write

As a <role>, draft <deliverable> for <audience> with <criteria>, ≤<length>. Output as <format>.

Summarize

Summarize for execs (≤120 words) with 3 key risks and 2 decisions; use bullets.

Extract

From the text, return a table with columns A, B, C; only exact matches.

Plan

Create a 5‑step plan with owners/dates/risks; finish with open questions.

Critique

Critique the draft against explicit criteria; list fixes by section before revising.

Tone‑shift

Rewrite in a respectful, confident tone for a frustrated customer; keep to 120–150 words.

These cover most daily use cases. Copy into your notes/macros.
